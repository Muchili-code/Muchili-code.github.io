# 高级程序设计

这门课......，确实是学到了些东西，但是这门课最大的缺点就是考核方式。主要是让学生学习面向对象编程的Java语言，却几乎没有任何练习，对比大一的C语言简直天壤之别的培养方式。

前期讲 Java 的基本语法，中期讲 Java 的一些进阶类/常用类，后期主要讲Java正则和设计模式。

**学习建议**

<span class="heimu" title="这是一个秘密">上课不用听，老师讲不如自己看 PPT（当然有些课件完全没有看的必要，自己斟酌），有不懂的问 AI 。让 AI 帮忙总结内容、提问、答疑解惑，效率比老师上课要好多了。作业的话提前完成认真做，占比很大的。<br>最后的考试要手搓代码，建议平时学习的时候也重点关注基本语法和相关函数的使用，考前再搓一下。<br>提醒一下哪怕看其他教程学完了，课件也得留意下，因为老师的考试肯定是根据课件来的，此外注意老师最后一节课的划重点。</span>
