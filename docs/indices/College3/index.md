# 碎碎念
作为BME的学生，大三的课程其实在理论上与专业科研是十分重要的。不过在我的个人学习上感觉...课程确实重要，但是学院的教学管理和大部分的教学质量都难以与十分重要的课程匹配。要不说 BME 是超级水的呢。其实课程内容很广，但是奈何学时限制，老师只能每个点都浅浅地讲解，再加上老师基本上不怎么布置作业，所以一学期下来我基本上没学到什么有用的知识点。
