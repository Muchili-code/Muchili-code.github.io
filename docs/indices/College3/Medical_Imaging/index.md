# 生物医学成像技术

这门课倒是有一些可以讲的东西。

首先，这门课已经是 BME 大三上为数不多的真的可以学到知识的课，哪怕知识皮毛。通过这门课，你可以基本上对现在的医学成像技术及其原理有基本的认识。

这门课的知识点分成以下几个板块：`XRay与CT`，`PET与SPECT`，`US/超声`，`Optics/光学`，`MRI`。每个板块由对应的老师讲三周的课。每个老师有各自的特色，大家也会适应老师的特色

此外，今年还进行了课改，主要是效仿 约翰霍普金斯的Prince讲课内容，同时考试形式也改成半开卷，对于考题难度的设置没那么难，

**学习方法**

<span class="heimu" title="你知道的太多了">
CT 和 MRI 还是好好听，zxjj讲的真的不错，而且 CT 和 MRI 最后都涉及到比较关键的计算，这个原理搞懂还是很有帮助的；
<br>
PET部分，hgf老师讲课......信息密度比较低，应该说他想讲好，但是显然没讲好。上课可以不听，当然能接受老师的讲课的话也可以听。
<br>
US部分，zyx讲课毫无疑问，别听，人在那里就行，听课万万不可，听课就是受罪。找学长学姐的资料学习会更有帮助
<br>
光学部分没怎么听，不过老师应该是不错的，诊疗的同学也说讲课比较细致。
</span>
