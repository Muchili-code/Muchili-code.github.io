# 误差理论与数据处理

这门课，学时很紧。本来可以16周每周讲2节课，却安排了8周每周讲4节课。

**学习方法**：没什么好说的，一门理论不是很难的课，重点是理解、运用 + 背公式 + 计算。上课听讲，课后作业和教程例题认真做，及时总结复习。这门课有点绩杀，基本上不怎么给过程分应该，因为小分点很多，答案错了就是错了没有过程分的。
