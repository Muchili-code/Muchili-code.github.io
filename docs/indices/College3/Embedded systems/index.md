# 嵌入式系统

这门课，一样的，讲的东西太多而不精。本来就是一门需要好好钻研的课，却不给什么作业。学生也很难自觉学习。至于最后的大作业，基本上都是ai完成的。

**学习建议**(Secret🤐)

<span class="heimu" title="你知道的太多了">前面的课程基本上都可以不听，考试的时候花一天时间背诵就行了。。。后面gcg老师讲操作系统这部分内容的时候，看一下课件，对进程、线程有基本的概念，且能看懂老师给出的例程的输出就行。此外，gcg老师讲线程编程那块好好听，这部分考试基本上都会考到。然后记得最后一节课要听，老师会划重点</span>
